Extension Number Command
========================

This command is for quickly looking up your colleagues extension numbers and calling them (Hey! Talk more with them! Don't just chat! `:-)`).

## Usage

    /ext [name]

Where `name` is optional. If omitted, it will show all extension numbers available.

## Configuration

Rename `config.example.json` to `config.json` and edit it to your needs.
Note: `config.json` is gitignored so your particularities will never saved to git.

## Usage examples

    /ext Luiz
    /ext
    /ext cto ---> This could be an alias for a real person
    /ext meeting-room
